# Curate - Issue Tracker

This repository is for tracking bugs, feature requests, and other issues for **Curate**, an Android app for rating and curating your favorite media.

## About Curate

Curate helps you organize and rate movies, TV shows, video games, and books using:
- **Head-to-head comparisons** - Compare two items to build your personal rankings
- **Simple ratings** - Quick 1-10 scores for individual items
- **Cloud sync** - Keep your ratings synchronized across devices
- **Offline support** - Works without an internet connection

## Reporting Issues

### Bug Reports
Found a bug? Please include:
- **Device & Android version** (e.g., Pixel 7, Android 14)
- **Steps to reproduce** the issue
- **Expected behavior** vs **actual behavior**
- **Screenshots** if applicable

### Feature Requests
Have an idea? Tell us:
- **What feature** you'd like to see
- **Why** it would be useful
- **How** you envision it working

### General Guidelines
- Search existing issues before creating a new one
- Use clear, descriptive titles
- Be respectful and constructive
- One issue per report
